# triangle

## 問題
以下のように、与えられた三角形の3辺の長さから、三角形の形を表示するプログラムを作成してください。

```ruby
$ ruby triangle.rb 2, 3, 4
不等辺三角形ですね！

$ ruby triangle.rb 2, 2, 1
二等辺三角形ですね！

$ ruby triangle.rb 1, 1, 1
正三角形ですね！

$ ruby triangle.rb 1, 2, 3
三角形じゃないです＞＜
```
## セットアップ
```
$ git clone https://github.com/oshino-taihei/triangle.git
$ gem install bundler
$ rbenv local 2.1.1
$ bundle install
```
